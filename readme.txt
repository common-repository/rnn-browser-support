== Description ==
Easy way to inform people that their browser might not support every element on the site. 

It currently only works with Internet Explorer and does not support any sort of version checking.

= Features to come =
* Support for multiple browsers

== Installation ==
= Manual Install =
Unzip rnn-browser-support.zip and upload the unzipped folder to /wp-content/plugins/ directory of your WordPress installation.

= Via Plugin Backend =
Upload rnn-browser-support.zip from your plugin panel (/wp-admin/plugin-install.php?tab=upload).

= Via Plugin Search =
Search for \"rnn browser support\" and install the displayed result from your plugin panel (/wp-admin/plugin-install.php)

= Activation =
Remember to activate RNN Browser Support after installation.
